HOW TO USE 'MasterNode-test-0.0.1.jar'

If you want to run this jar file to do clustering of pdb files,
you have to execute the following steps:

-



- enter the pdb file location as a command line:
    java -jar MasterNode-test/build/libs/MasterNode-test-0.0.1.jar --pdb.file.location=file:/your/pdb/file/location/*.gz
    java -Dpdb.file.location=file:/your/pdb/file/location/*.gz -jar MasterNode-test/build/libs/MasterNode-test-0.0.1.jar
